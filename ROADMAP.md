- polish public template structure

- add profile-level demos and validation coverage
- refine MCP stub behavior and demo outputs

## v0.2.x
- add safer profile packaging helpers

## v0.3.x
- add optional CI for validation scripts
- add richer synthetic demo bundles
- improve profile migration guides
- document safer real-backend integration patterns
- deepen protocol capability packs for Modbus / OPC UA / HMI / PLC
- normalize canonical agent roles and output contracts
- connect capability packs to reusable examples and script templates

## Non-goals
- direct production hardware control
- claims of certification or safety approval
- shipping real customer artifacts or private topology data

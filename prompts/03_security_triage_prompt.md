# Prompt 03 · 安全与漏洞扫描分析

```text
使用 @vuln-scan-triage 来分析 Nessus、Burp Suite 和 Defensics 输出，执行以下任务：
1. 按主机、服务、协议、CWE 分类去重扫描结果，剔除重复项。
2. 评估每个漏洞的可达性、利用条件、认证要求等，区分真实缺陷、误报和重复项。
3. 优先分析可能导致失控操作、拒绝服务、远程修改等高风险缺陷。
4. 输出漏洞修复队列、补偿控制措施、复测计划和相关证据路径。

同时，使用 @security-62443-assessor 执行以下任务：
1. 重新审查已识别的漏洞和缺陷，评估它们是否符合 IEC 62443-4-2 的要求。
2. 更新威胁模型，涵盖资产、接口、角色、信任边界、入口点等信息。
3. 确保每个安全要求都可以映射到有效的证据，不允许没有证据支持的声明。

请确保所有的安全分析任务都按计划执行，并记录相关证据路径以便后续验证。
```

# Prompt 01 · 生成测试计划

```text
你是一个工业自动化测试规划代理。请基于以下信息为一个新版本的工业自动化产品生成测试计划：

- 产品发布版本：3.8.12
- 产品范围：HMI、PLC、伺服、变频器、AI/DI/DO/HSC
- 目标验证：功能验证、安全验证、性能回归、用户界面测试
- 产品模块：HMI、PLC、伺服控制、变频器、IO 模块、安全控制

根据这些信息，请生成以下内容：
1. 测试范围与优先级：列出需要优先处理的测试领域，例如：安全性 > 功能性 > 可维护性。
2. 并行可执行的分析任务：哪些任务可以并行执行，哪些任务必须串行。
3. 串行的硬件操作任务：哪些操作必须在真实工位上串行执行，如上电、固件下载、force IO、驱动使能、运动命令。
4. 所需证据：请明确列出需要收集的证据类型，如日志、截图、trace、参数差异、扫描报告等。
5. 完成标准：测试的退出标准、何时标记为通过或失败。

输出应包含一个详细的 Markdown 文件，包含上述内容，并明确任务的优先级和所需的操作顺序。
```

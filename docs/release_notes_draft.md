# Release Notes Draft

## v0.1.0 - public template seed

This release publishes the first public, demo-safe version of the GitHub Copilot industrial multi-agent template repository.

### Included
- shared multi-agent project skeleton
- HMI / PLC / Security profile packs
- advanced prompt pack
- MCP stub servers and demo helpers
- validation scripts
- report, evidence, and runbook templates
- root and profile-level demo flows
- public-facing documentation and packaging guidance

### Safety note
This repository is a template and demonstration pack. It does not enable production-safe hardware control out of the box.
Real bench integration should remain human-controlled and protected by server-side policy.
